A command with fzf-like UI to quickly search Wikipedia articles and open it in your browser.

</div>

## Usage

```
Usage:
  fzwiki [OPTIONS] QUERY...

Application Options:
  -o, --open  open URL with webbrowser
  -l, --lang= wikipedia language

Help Options:
  -h, --help  Show this help message
```
